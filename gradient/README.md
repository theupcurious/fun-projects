# Gradient

A color perception game. Each round shows a grid of nearly identical tiles with exactly one odd tile — find it before you run out of lives.

## Features

- Difficulty ramp based on level and grid density
- Speed and streak score bonuses
- Wrong-answer feedback with correct-tile reveal
- Shareable result card
- Local high score persistence

## Stack

- Next.js 15, React 19, TypeScript
- Tailwind CSS
- Framer Motion
- `html-to-image`

## Getting Started

```bash
npm install
npm run dev
```

`npm run build` produces a static export in `out/`.
