# Vibe

Design language generator for vibe coders. Define your palette, typography, and component style — then export a structured prompt for Cursor, Claude, or ChatGPT.

## Features

- 18 mood presets across Professional, Creative, and Bold categories
- Live preview template that updates instantly as you adjust controls
- Color palette editor with 8 named tokens
- Typography controls — curated Google Fonts, weight, size, letter-spacing, heading transform
- Shape & spacing — border radius, density, shadow, border weight
- Component style selectors — buttons, cards, inputs, links
- Export as Markdown, system prompt, or `.cursorrules`
- Shareable URL — full configuration encoded in the URL hash

## Stack

- Next.js 16, TypeScript
- Tailwind CSS v4
- shadcn/ui
- react-colorful
- Google Fonts

## Getting Started

```bash
npm install
npm run dev
```
