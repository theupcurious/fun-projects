# Fun Projects

A collection of open source tools and experiments by [Upcurious](https://upcurious.co).

---

## Projects

### [Gradient](gradient/)

A color perception game. Each round shows a grid of nearly identical tiles with exactly one odd tile — find it before you run out of lives. Features a difficulty ramp, speed and streak bonuses, local high score persistence, and a shareable result card.

**Stack:** Next.js 15, React 19, Tailwind CSS, Framer Motion, TypeScript

---

### [OG Previewer](og-previewer/)

Paste any public URL to see how it will appear when shared on X, WhatsApp, and Slack. Includes heuristic diagnostics for common metadata issues and a raw meta tag inspector.

**Stack:** Next.js 16, Tailwind CSS v4, shadcn/ui, cheerio, Zod

---

### [Vibe](vibe/)

Define a distinctive design language — colors, fonts, spacing, component style — preview it live on a realistic template, and export a structured prompt you can paste into Cursor, Claude, or ChatGPT to override the generic AI aesthetic.

**Stack:** Next.js 16, Tailwind CSS v4, shadcn/ui, react-colorful, TypeScript
